# Portfolio Web
Professional Portfolio Website (assignment submission for Internet Programming)

# Scenario: 
You are a budding web developer seeking to showcase your skills and build a professional online 
presence. To accomplish this, you have decided to create a personal portfolio website. The 
website will serve as a platform to display your web development projects, skills, and contact 
information. 
 
# Requirements: 

1. HTML Structure: 
• Create an HTML document that outlines the structure of your portfolio website. 
• Use appropriate HTML tags to create a header, navigation menu, main content section, 
and footer. 
• Ensure the HTML is semantically structured, with proper headings and paragraphs. 
 
2. CSS Styling: 
• Apply CSS styles to your website to enhance its visual appeal. 
• Implement a consistent color scheme and typography throughout the website. 
• Use CSS selectors to style specific elements, such as headings, links, and images. 
• Utilize CSS for layout and positioning to ensure a visually pleasing design. 
 
3. Any appropriate content inside the poftfolio.  
 
4. Skills and Achievements: 
• Design a section where you can list your technical skills and achievements related to web 
development. 
• Utilize HTML lists or tables for clear organization. 
 
 
5. Contact Information: 
• Include a "Contact" section with your email address and social media links. 
• Create clickable links that open your email client and social media profiles in new tabs. 
 
6. Responsive Design: 
• Ensure your website is responsive and displays correctly on both desktop and mobile 
devices. 
• Implement CSS media queries to adapt the layout and styling for various screen sizes.
